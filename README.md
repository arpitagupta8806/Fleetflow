# FleetFlow App

A mockup of a Fleet & Logistics Management mobile app built with HTML, CSS, and JavaScript.

## Features

- **AI-Powered Fleet Management**: Live tracking, route optimization, predictive maintenance
- **Interactive Screens**: Splash, Home, Tracking, Routes, Fuel Management, Maintenance, Compliance, Alerts, Profile
- **Responsive Design**: Scales to fit different screen sizes
- **Modern UI**: Dark theme with smooth animations and transitions

## Project Structure

```
Fleetflow/
├── index.html          # Main HTML file
├── styles.css          # All styles
├── script.js           # JavaScript functionality
└── README.md           # This file
```

## How to Run

1. Open `index.html` in a web browser
2. Click on buttons to navigate between screens
3. The app simulates a mobile phone interface

## Technologies Used

- HTML5
- CSS3 (with CSS Variables)
- JavaScript (ES6)
- Tabler Icons (via CDN)
- Google Fonts (Sora and DM Sans)
